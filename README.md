# MiniProjectApi
API RESTful desarrollada en .NET Core con autenticación JWT y desplegado en Azure. Proporciona endpoints para gestionar usuarios y autenticar solicitudes.
## Características:

* Autenticación mediante JWT.
* Conectividad a base de datos SQL Server.
* Alojada como App Service en Azure.

## Tecnologías Utilizadas:

* .NET Core
* Entity Framework Core
* JWT Authentication
* Azure App Services y Azure Database SQL.
